#  Игра "Угадай число "

##  Описание
Веб-приложение, в котором пользователь должен угадать случайное число.  
Игра даёт подсказки ("больше" или "меньше"), считает попытки и время,  
а также сохраняет лучшие результаты.

---

##  Функционал

- Генерация случайного числа
- Выбор уровня сложности:
  - Легко (1–50)
  - Нормально (1–100)
  - Сложно (1–1000)
- Подсказки игроку
- Подсчёт количества попыток
- Таймер
- Таблица рекордов (ТОП-10)
- Сохранение результатов в localStorage
- Анимации интерфейса

---

##  Используемые технологии

- HTML
- CSS
- JavaScript 
- localStorage
